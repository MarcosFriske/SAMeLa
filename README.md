# ValidaLattes
 Pasta principal do projeto é SiteLattesPrototipo, arquivo que deve ser executado (servidor) é o app.py.

 Imagens showcase são prints das telas do site no estágio atual.
 Qualis é a base qualis que subimos no nosso banco de dados.
